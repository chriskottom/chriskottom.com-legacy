Start with the Fundamentals

Learning to use a simple tool effectively involves no small amount of art, and that's why I'm writing The Minitest Cookbook - to help developers of all experience levels to master the techniques needed to write clear, maintainable tests that support them as they try to produce better applications. All of this starts with a foundation of basic testing tactics.

  • Recognize the testing that's needed for your application
  • Structure your tests and specs effectively
  • Manage code shared between tests
  • Use mocking, stubbing, and other kinds of test doubles
  • Spot poorly performing test code early
  • Use supporting gems and plugins to trick out your stack
